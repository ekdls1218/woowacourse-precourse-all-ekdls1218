# 📝 우아한테크코스 프리코스 학습 기록

프리코스 과정을 진행하며 **기능 구현 → 테스트/리팩토링 → 구조 개선**의 사이클을 경험했습니다.   
문제를 단순히 해결하는 데서 끝나지 않고, **코드를 왜 이렇게 작성해야 하는가**를 고민하며 개발 역량을 키웠습니다.

이 저장소는 그 과정에서 수행한 **4개의 과제와 주요 학습 내용**을 정리한 기록입니다.

- 스트라이크/볼 판정, 승/패 판정 및 재시작/종료 기능, 입출 구현
- 테스트 작성

### 🔹 4주차) 크리스마스 프로모션
이메일 형식의 기능 요구 사항에 맞춰 주어진 문제의 내용을 통해 유추하고 스스로 판단해 구현
고객들이 식당에 방문할 날짜와 메뉴를 미리 선택하면 이벤트 플래너가 주문 메뉴, 할인 전 총 주문 금액, 증정 메뉴, 혜택 내역, 총 혜택 금액, 할인 후 예상 결제 금액, 12월 이벤트 배지 내용을 보여주기를 기대
- [과제 요구 사항](https://github.com/ekdls1218/woowacourse-precourse-all-ekdls1218/blob/main/javascript-christmas-6/README.md)
- [기능 목록](https://github.com/ekdls1218/woowacourse-precourse-all-ekdls1218/blob/main/javascript-christmas-6/docs/README.md)
- 사용자 입력 검증 및 예외 처리
- 이벤트, 이벤트 적용, 계산, 입출력 기능 구현
- 단위 테스트 작성
